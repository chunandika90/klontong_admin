Cara Menjalankan UAT

# 1. Pastikan Docker Desktop sedang berjalan
# 2. Dari root project:
docker compose up -d

# 3. Tunggu semua container healthy (±2 menit)
# 4. Buka browser: http://localhost:4000
# 5. Login dengan: admin@klontong.com / admin123

Flow UAT Manual

Login Buka http://localhost:4000/auth/login → masuk dengan akun admin
Dashboard Cek 4 kartu stats + bar chart + 2 tabel (Stok Menipis & Produk Terbaru)
Klik Kartu Stats Verifikasi redirect ke filter yang benar di halaman produk
Product List Klik "Produk" di navbar → test search, filter kategori, filter status, sort, pagination
Create Produk Klik "Tambah Produk" → isi form di modal → simpan → verifikasi list refresh
Edit dari List Klik Edit di baris produk → ubah data di modal → simpan → verifikasi data berubah
Detail & Edit dari Detail Klik Detail → halaman detail → klik "Edit Produk" → ubah → simpan → auto reload
Archive Produk Klik Arsip → konfirmasi → produk hilang dari list (soft delete)
Register Akun Baru Buka /auth/register → daftar → langsung masuk Dashboard
API Docs (Swagger) Buka http://localhost:4001/api/docs → eksplorasi semua endpoint
